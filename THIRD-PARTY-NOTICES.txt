# Third-Party Notices

Provenance is licensed under the Apache License, Version 2.0 (see [`LICENSE`](LICENSE)
and [`NOTICE`](NOTICE)). It redistributes third-party open-source components in three
distributed artifacts:

- **Recorder** (`packages/recorder`) — packaged as a VSIX, bundling its runtime dependencies.
- **Analyzer** (`packages/analyzer`) — built by Vite into a static SPA that includes React
  and its other runtime dependencies.
- **Server** (`packages/server`) — a Node.js process that ships its runtime dependencies at
  deploy time.

Each also pulls in the repo's internal libraries (`packages/log-core`, `packages/shared`,
`packages/analysis-core`), which are not published separately; their dependencies are
folded into the artifact that bundles them.

The tables below list the full transitive set of production dependencies resolved for each
artifact (`npm ls --workspace=packages/<name> --omit=dev --all`). Build/test-only
`devDependencies` are excluded, as are optional dependencies that were never installed
(e.g. the alternate database drivers `drizzle-orm` supports). Each license and copyright
line comes from the installed package's `package.json` and its `LICENSE`/`NOTICE`/`COPYING`
file where present. Full license text for each package is in its own
`node_modules/<pkg>/LICENSE` or on [npmjs.com](https://www.npmjs.com/). None of the
Apache-2.0 dependencies ship a `NOTICE` file.

Rows marked **inferred** ship no `LICENSE` or `author` of their own; the copyright shown
was copied from a sibling package in the same npm scope that does ship one. Rows marked
**UNVERIFIED** could not be attributed to a copyright holder — see the per-section notes.

**Native platform binaries:** `@node-rs/argon2-<platform>` and `@node-rs/bcrypt-<platform>`
are per-platform optional dependencies; npm installs only the one matching the build
machine. This document was generated on `darwin-arm64` and lists the `-darwin-arm64`
variants; a Linux deployment resolves the equivalent `-linux-x64-gnu` package under the
same license and copyright holder.
