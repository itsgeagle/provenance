# Bundled fonts

Self-hosted for the `/architecture` map. Both are licensed under the SIL Open
Font License 1.1 (see `OFL.txt`).

- **Space Grotesk** — Florian Karsten. Display / titleblocks.
- **IBM Plex Mono** — IBM / Bold Monday. Instrument readouts, labels, node addresses.

Files are the latin subset (weights 400–700), pulled from the Fontsource CDN.
Regenerate with the curl block in the repo's font-update notes if a weight is added.
